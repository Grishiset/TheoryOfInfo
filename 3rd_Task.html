Шифротекст (64 бит)
Функция f(R, K):
1. Расширяющая перестановка E: 32 → 48 бит
2. XOR с подключом Kᵢ (48 бит)
3. 8 S-блоков: 48 → 32 бит (нелинейность)
4. Перестановка P: 32 → 32 бит

Генерация ключей (Key Schedule):
64-битный ключ → PC-1 (56 бит) → 16 раздвиговых циклов → PC-2 (48 бит) на раунд

ECB: каждый блок шифруется независимо — Cᵢ = E(Pᵢ)
